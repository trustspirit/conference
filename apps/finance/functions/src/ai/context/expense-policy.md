## 지출 한도
- 일반 경비: 건당 한도 없음 (프로젝트 예산 범위 내)
- 교통비: 실비 정산 (대중교통 기준, 카카오 모빌리티 API로 거리 자동 계산)
- 식비: 1인당 1만원 이내
- 숙박비: 사전 승인 필요

## 필수 서류
- 모든 지출에 영수증 첨부 필수
- 교통비: 출발지/도착지 입력 시 이동 경로가 자동으로 기록됨
- 10만원 이상: 세금계산서 또는 카드 영수증 필수
- 현금 영수증도 인정됨

## 결재 프로세스
1. **재무 담당 검토**: 영수증 확인, 예산 코드 확인, 금액 확인
2. **결재권자 승인**: 최종 결재

### 위원회별 결재 라인
- **운영위원회**: 운영 재무담당(finance_ops) 검토 → 운영 결재권자(approver_ops) 승인
- **준비위원회**: 준비 재무담당(finance_prep) 검토 → 준비 결재권자(approver_prep) 승인
- **60만원 초과**: 상위 결재권자(국장/본부장급) 승인 필요
  - 운영위: session_director 승인
  - 준비위: logistic_admin 승인
- **executive(조직장)**: 모든 위원회, 금액 제한 없이 승인 가능

## 반려 사유 (주요)
- 영수증 누락 또는 내용 불일치
- 예산 코드 오류 (잘못된 예산 코드 선택)
- 한도 초과
- 중복 청구 (동일 건 재청구)
- 날짜 오류

## 예산 관리
- 각 프로젝트별 총 예산과 예산 코드별 예산이 설정되어 있습니다.
- 대시보드에서 현재 예산 사용 현황을 확인할 수 있습니다.
- 예산 소진률이 높을 경우 경고가 표시됩니다.
