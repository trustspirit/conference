## 경비 정책 및 규정

이 문서는 컨퍼런스 경비 청구와 관련된 모든 정책, 한도, 필수 서류, 결재 프로세스를 설명합니다.

---

### 1. 지출 한도

| 구분              | 한도                  | 비고                               |
| ----------------- | --------------------- | ---------------------------------- |
| 일반 경비         | 건당 한도 없음        | 프로젝트 예산 범위 내에서 가능     |
| 교통비 (대중교통) | 실비 정산             | 실제 지출 금액                     |
| 교통비 (자가용)   | 거리 기반 자동 정산   | km당 단가 × 거리 × 이동배수        |
| 식비              | 1인당 1만원 이내      | 초과 시 사유 기재 필요             |
| 숙박비            | 사전 승인 필요        | 사전에 결재권자의 승인을 받아야 함 |
| 회의 식비         | 1인당 1만원 이내      | 참석자 목록 기재 권장              |
| 다과/간식비       | 프로젝트 예산 범위 내 | 영수증 필수                        |

---

### 2. 필수 서류

#### 기본 요건

- **모든 지출**: 영수증 첨부 필수 (PNG, JPG, PDF / 최대 **750KB**/파일)
- **현금 영수증**도 유효한 영수증으로 인정됩니다.
- 하나의 청구서에 여러 영수증 파일을 첨부할 수 있습니다.

#### 금액별 추가 요건

| 금액        | 필요 서류                        |
| ----------- | -------------------------------- |
| 10만원 미만 | 일반 영수증 (간이영수증 가능)    |
| 10만원 이상 | 세금계산서 또는 카드 영수증 필수 |

#### 교통비 전용 요건

- **출발지/도착지 입력 필수**: 앱에서 주소를 검색하여 입력하면 카카오 모빌리티 API를 통해 이동 경로와 거리가 자동으로 기록됩니다.
- 별도의 교통비 영수증이 없어도, 시스템에서 자동 생성된 경로 기록이 증빙으로 사용됩니다.

**자가용 이용 시 — 거리 기반 자동 계산:**

- 계산 공식: **환급금 = 거리(km) × km당 단가(원) × 이동 배수**
- 이동 배수: 왕복 = 2, 편도 = 1
- 기본 km당 단가: **₩250** (프로젝트 설정에서 변경 가능)
- 거리는 카카오 모빌리티 API가 실제 도로 경로 기준으로 자동 산출합니다.
- 금액은 시스템이 자동으로 계산하여 입력하므로, 사용자가 직접 금액을 입력할 수 없습니다.
- 예시: 서울→수원 30km 왕복 = 30km × ₩250 × 2 = **₩15,000**
- 예시: 서울→인천 40km 편도 = 40km × ₩250 × 1 = **₩10,000**

**대중교통 이용 시 — 실비 정산:**

- 실제 지출한 금액을 직접 입력합니다.
- 영수증(교통카드 내역, 승차권 등)을 첨부합니다.

#### 통장 사본

- 통장 사본 파일: PNG, JPG, PDF / 최대 **800KB**
- 통장 사본은 **프로필 페이지에서 미리 등록**하거나, **첫 신청서 작성 시 인라인으로 업로드**할 수 있습니다. 인라인 업로드 시 프로필에 자동 저장됩니다.
- 통장 사본이 프로필에도 없고 신청서에서도 업로드하지 않으면 제출이 불가합니다.
- **업체 신청서의 경우**: 프로필의 개인 통장 사본 대신, 신청서 작성 시 **업체 통장 사본을 별도로 첨부**해야 합니다. 업체 통장 사본은 프로필에 저장되지 않고 해당 신청서에만 연결됩니다.

### 3. 결재 프로세스 (2단계)

경비 청구는 **2단계 결재**를 거칩니다.

#### 1단계: 재무 담당 검토 (대기중 → 검토완료)

- **담당자**: 운영 재정담당 (운영위만), 준비 재정담당 (모든 위원회)
- **확인 항목**: 영수증 첨부 여부, 예산 코드 정확성, 금액 일치, 지출 한도 준수
- **결과**: 검토 완료 또는 반려

#### 2단계: 결재권자 승인 (검토완료 → 승인)

- **담당자**: 해당 위원회의 결재권자
- **확인 항목**: 지출 적정성, 예산 범위 내 여부
- **결과**: 승인 또는 반려

#### 위원회별 결재 라인

**운영위원회 청구:**
| 금액 | 검토자 | 승인자 |
|------|--------|--------|
| 60만원 이하 | 운영 재정담당 또는 준비 재정담당 | 운영 결재권자 |
| 60만원 초과 | 운영 재정담당 또는 준비 재정담당 | 운영 위원장, 대회장, 또는 시스템 관리자 |

**준비위원회 청구:**
| 금액 | 검토자 | 승인자 |
|------|--------|--------|
| 60만원 이하 | 준비 재정담당 | 준비 결재권자 |
| 60만원 초과 | 준비 재정담당 | 준비 위원장, 대회장, 또는 시스템 관리자 |

> **참고**: 60만원은 기본값이며, 프로젝트 설정에서 변경 가능합니다.

#### 특수 결재 규칙

- **본인 청구 본인 검토/승인 불가**: 자기가 제출한 청구서는 자기가 검토하거나 승인할 수 없습니다.
- **위원장 청구**: 운영 위원장 또는 준비 위원장이 제출한 청구는 대회장 또는 시스템 관리자만 승인 가능합니다.
- **대회장**: 모든 위원회, 금액 제한 없이 승인 가능합니다.
- **강제 반려**: 이미 승인된 청구도 준비 재정담당 또는 시스템 관리자가 강제 반려할 수 있습니다. 단, 정산완료된 건은 강제 반려할 수 없습니다.

### 4. 예산 코드별 사용 가이드

각 지출 항목에 맞는 예산 코드를 선택해야 합니다. 잘못된 예산 코드 선택은 반려 사유가 됩니다.

| 세부 항목 | 설명                              |
| --------- | --------------------------------- |
| 숙박      | 호텔, 게스트하우스 등 숙박 비용   |
| 기타 숙박 | 위에 해당하지 않는 숙박 관련 비용 |
| 시설 대여 | 행사장, 회의실 등 시설 대여 비용  |
| 장비 대여 | 음향, 조명 등 장비 대여 비용      |
| 기타 시설 | 위에 해당하지 않는 시설 관련 비용 |

#### 5110 - 교통비

| 세부 항목      | 설명                          |

| 참가자 교통    | 참가자의 이동 교통비          |

### 5. 주요 반려 사유 및 대처 방법

| 반려 사유                     | 설명                                                 | 대처 방법                                     |
| ----------------------------- | ---------------------------------------------------- | --------------------------------------------- |
| 영수증 누락                   | 영수증이 첨부되지 않았거나 일부 항목에 영수증이 없음 | 영수증을 첨부하여 재제출                      |
| 영수증 내용 불일치            | 영수증 금액과 청구 금액이 다름                       | 금액을 확인하고 수정하여 재제출               |
| 예산 코드 오류                | 지출 내용에 맞지 않는 예산 코드를 선택함             | 올바른 예산 코드를 선택하여 재제출            |
| 한도 초과                     | 식비 1인 1만원 등 한도를 초과함                      | 한도 내로 금액 수정 또는 초과 사유 기재       |
| 중복 청구                     | 이미 청구된 건과 동일한 내용으로 다시 청구함         | 중복 확인 후 취소하거나 사유 기재             |
| 날짜 오류                     | 지출 날짜가 잘못 입력됨                              | 올바른 날짜로 수정하여 재제출                 |
| 10만원 이상 세금계산서 미첨부 | 10만원 이상 지출에 세금계산서/카드영수증이 없음      | 세금계산서 또는 카드 영수증을 첨부하여 재제출 |
| 숙박비 사전 승인 미확보       | 숙박비를 사전 승인 없이 청구함                       | 사전 승인을 받은 후 재제출                    |

> **반려된 청구의 재제출**: '내 청구' 목록에서 반려된 청구를 선택 → '재제출' 클릭 → 반려 사유 참고하여 수정 → 다시 제출. 재제출 시 이전 내용이 자동으로 채워지며, 원본과 달라진 점이 최소 한 곳 이상 있어야 재제출이 가능합니다.

### 6. 예산 관리 정책

- 각 프로젝트별 **총 예산**과 **예산 코드별 예산**이 설정되어 있습니다.
- 대시보드(준비 재정담당, 상위 결재권자, 시스템 관리자 접근 가능)에서 현재 예산 사용 현황을 확인할 수 있습니다.
- **예산 집계 기준**: **검토완료, 승인, 정산완료** 상태의 청구 금액이 예산 사용으로 집계됩니다. 대기중, 반려, 강제반려, 취소된 청구는 예산에 포함되지 않습니다.
- **예산 경고**: 프로젝트 설정의 경고 임계값(기본 85%)을 초과하면 대시보드에 경고 배너가 표시됩니다.
- **예산 초과 100%**: 예산이 완전히 소진된 경우 별도 경고가 표시됩니다.
- 예산 초과 시에도 청구 제출은 가능하지만, 검토/승인 단계에서 반려될 수 있습니다.

### 7. 이메일 알림 정책

| 시점                             | 수신자                  | 내용                                             |
| -------------------------------- | ----------------------- | ------------------------------------------------ |
| 새 청구 제출 시                  | 해당 위원회 재무 담당자 | 새 청구가 접수되었다는 알림                      |
| 검토 완료 시                     | 해당 위원회 결재권자    | 승인 요청 알림 (위원장 청구의 경우 대회장에게만) |
| 승인 시                          | 청구 제출자             | 최종 승인되었다는 알림                           |
| 반려 시                          | 청구 제출자             | 반려되었다는 알림 (반려 사유 포함)               |
| 강제 반려 시                     | 청구 제출자             | 승인 후 강제 반려되었다는 알림                   |
| 매주 일요일 오전 9시 (한국 시간) | 각 역할별 담당자        | 미처리 청구 현황 다이제스트                      |

### 8. 파일 업로드 제한

| 파일 종류 | 지원 형식     | 최대 크기  | 비고                                            |
| --------- | ------------- | ---------- | ----------------------------------------------- |
| 영수증    | PNG, JPG, PDF | 750KB/파일 | 여러 파일 첨부 가능                             |
| 통장 사본 | PNG, JPG, PDF | 800KB      | 1개만 저장 (새로 업로드 시 기존 파일 자동 삭제) |
| 경로 지도 | JPEG          | 자동 생성  | 자가용 교통비 항목 시 시스템이 자동 캡처 (OpenStreetMap 기반) |
