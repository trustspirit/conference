## 관리자 메뉴 (admin 역할 이상)
- **청구 관리**: 모든 청구 내역 조회, 검토, 승인/반려 처리
- **대시보드**: 예산 현황, 승인 추세, 예산 코드별 분석, 월별 트렌드 차트
- **사용자 관리**: 사용자 역할 변경, 위원회 배정
- **영수증 관리**: 전체 영수증 조회, 일괄 다운로드
- **정산 관리**: 승인된 청구 선택 → 정산 보고서 생성 → PDF 출력
- **프로젝트 관리**: 프로젝트 생성, 예산 설정, 멤버 관리, 프로젝트 삭제(30일 복구 가능)

## 역할 체계
- **user**: 일반 사용자 (청구 제출만 가능)
- **finance_ops**: 운영위 재무담당 (운영위 청구 검토)
- **approver_ops**: 운영위 결재권자 (운영위 청구 승인, 60만원 이하)
- **finance_prep**: 준비위 재무담당 (준비위 청구 검토 + 정산 처리)
- **approver_prep**: 준비위 결재권자 (준비위 청구 승인, 60만원 이하)
- **session_director**: 운영 국장 (운영위 청구 무제한 승인)
- **logistic_admin**: 준비 본부장 (준비위 청구 무제한 승인)
- **executive**: 조직장 (모든 위원회 무제한 승인)
- **admin**: 시스템 관리자 (모든 권한 + 사용자/프로젝트 관리)

## 정산 프로세스
1. 정산 메뉴에서 '새 정산'을 클릭합니다.
2. 승인된 청구 중 정산할 건을 선택합니다.
3. 수취인별로 자동 그룹핑됩니다.
4. 정산 보고서가 생성되며, 영수증과 경로 지도가 포함됩니다.
5. PDF로 출력하여 서명 후 보관할 수 있습니다.
