## 경비 청구 방법
1. 상단 메뉴에서 '신규 청구'를 클릭합니다.
2. 날짜, 소속 위원회(운영/준비), 지출 항목을 입력합니다.
3. 예산 코드를 선택하고 금액을 입력합니다.
4. 영수증을 첨부합니다 (PNG, JPG, PDF 지원).
5. 교통비의 경우 출발지와 도착지를 입력하면 카카오 모빌리티 API를 통해 자동으로 거리와 경로가 계산됩니다.
6. 모든 항목을 확인한 후 '제출' 버튼을 클릭합니다.

## 청구 상태
- **대기중(pending)**: 제출 후 재무 담당자의 검토를 기다리는 상태
- **검토완료(reviewed)**: 재무 담당자가 영수증과 내용을 확인 완료한 상태
- **승인(approved)**: 결재권자가 최종 승인한 상태
- **정산완료(settled)**: 송금이 완료된 상태
- **반려(rejected)**: 반려된 상태 (사유를 확인할 수 있음)
- **취소(cancelled)**: 본인이 직접 취소한 상태

## 내 청구 내역 확인
메뉴에서 '내 청구'를 클릭하면 본인이 제출한 모든 청구 내역과 현재 상태를 확인할 수 있습니다. 상태별 필터링과 정렬이 가능합니다.

## 은행 정보 등록
프로필 페이지에서 은행명, 계좌번호, 통장 사본 이미지를 등록해야 청구가 가능합니다. 통장 사본이 없으면 청구서 제출이 제한됩니다.

## 서명 등록
프로필 페이지에서 전자 서명을 등록할 수 있습니다. 서명은 정산 보고서에 포함됩니다.

## 청구서 재제출
반려된 청구서는 '내 청구' 목록에서 해당 청구를 선택한 후 '재제출' 버튼을 통해 수정하여 다시 제출할 수 있습니다.

## 프로젝트 선택
상단 헤더의 프로젝트 선택기를 통해 소속 프로젝트를 변경할 수 있습니다. 청구와 정산은 선택된 프로젝트 기준으로 처리됩니다.
